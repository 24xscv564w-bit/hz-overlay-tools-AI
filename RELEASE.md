Release v1.0.0 - Initial public release

Highlights
- Detects 120Hz and 144Hz and visually indicates both with distinct styles.
- 120Hz: animated rainbow + small nudge animation on change.
- 144Hz: red, italic, with glow for clear visibility.
- Resizable canvas to avoid chopped text; positions at bottom-right (configurable).

Assets
- hz_overlay.py (main script)

Notes
- This release is a distribution-ready package. The included script is scrubbed of any user-specific metadata.
- To build a Windows executable, use PyInstaller (see BUILD_EXE.md).
